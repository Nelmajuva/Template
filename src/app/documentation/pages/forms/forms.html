Forms

Form Control

Usa {{ '.form-control' }} para establecer el estilo del input dónde se ingresará la información. 

<div class="rounded-lg border border-gray-100 p-8 flex flex-col flex-wrap gap-3 items-start">
  <div class="input-group">
    <span class="input-group-prefix">
      <svg lucideChessQueen></svg>
    </span>
    <input type="text" class="form-control has-prefix" placeholder="name@example.com" />
  </div>
  <div class="input-group">
    <span class="input-group-prefix">
      <svg lucideChessQueen></svg>
    </span>
    <input type="text" class="form-control-solid has-prefix" placeholder="name@example.com" />
  </div>
  <div class="input-group">
    <input type="password" class="form-control has-suffix" placeholder="name@example.com" />
    <button type="button" class="input-group-suffix input-group-suffix-action">
      <svg lucideChessQueen></svg>
    </button>
  </div>
</div>

Error

Si se requiere mostrar que un campo presenta errores de información, se puede usar {{ '.input-error' }} para resaltar el campo de texto. 

<div class="rounded-lg border border-gray-100 p-8 flex flex-col flex-wrap gap-3 items-start">
  <div class="input-group input-error">
    <span class="input-group-prefix">
      <svg lucideChessQueen></svg>
    </span>

  </div>
  <div class="input-group input-error">
