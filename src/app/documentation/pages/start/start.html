Construye aplicaciones web modernas con velocidad

Un sistema basado en TailwindCSS con un diseño sólido, utilidades extensivas y componentes personalizados desarrollados internamente para escalar de forma consistente.

Directivas

Extienden el comportamiento del DOM, permitiendo manipular estructura y atributos de forma declarativa. Son fundamentales junto a componentes y servicios.

Componentes

Unidad base de la UI. Encapsulan estructura, lógica y estilos, promoviendo reutilización y separación de responsabilidades.

Pipes

Transforman datos directamente en el template sin alterar la lógica del componente, manteniendo la presentación desacoplada.

Servicios

Centralizan lógica de negocio, acceso a APIs y estado. Se integran mediante un sistema robusto de Dependency Injection.
